# INDLink Version 1.0 Release Notes

Welcome to **INDLink v1.0** — India's premier Creator Monetization and Link-in-Bio platform!

This major release aggregates months of development across 7 distinct phases, resulting in a robust, scalable, and enterprise-grade SaaS platform.

## 🚀 Key Features in v1.0

### 1. The Ultimate Link-in-Bio
*   Claim a unique username (`indlink.in/username`).
*   Drag & Drop link organization.
*   Deep analytics (Clicks, Unique Visitors, Geography, Device tracking).
*   Dynamic Custom Themes.

### 2. Creator Monetization
*   Native integration with **Razorpay** (Supports UPI, Cards, Netbanking).
*   Create and sell Digital Products (PDFs, Videos, Templates).
*   Secure digital downloads backed by Cloudflare R2 expiring links.
*   Zero platform fees on the Free tier.

### 3. Enterprise Infrastructure
*   Connect Custom Domains (`yourname.com` -> your INDLink profile).
*   Team Management & Role-Based Access Control (RBAC).
*   AI Profile Assistant powered by OpenAI `gpt-4o`.
*   Developer API (`/api/v1/public`) secured via hashed API Keys.
*   2FA Security (TOTP) and Active Session Management.

## ⚠️ Known Limitations
*   Wildcard Custom Domains currently require Cloudflare Enterprise or manual CNAME setups per user.
*   AI features are rate-limited to 5 requests/minute to prevent abuse.

## 🔮 Future Roadmap (v1.1+)
*   Affiliate Marketing System & Referrals.
*   Physical Products Storefront.
*   Stripe / PayPal Integration for global payments.
*   Webhooks for Creators (Send purchase events to Zapier/Make).
